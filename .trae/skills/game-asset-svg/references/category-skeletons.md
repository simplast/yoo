# 类别骨架

> 4px 网格，64×64 = 16×16 格 / 32×32 = 8×8 格。
> x / y / width / height 都按格数计算（每格 4px）。

## Tower 骨架（64×64）

```
y=0  ┌──────────────────────────────┐
     │  0   4   8  12  16  20  24  28  32  36  40  44  48  52  56  60
y=4  │       ╔══════════════════╗      │  ← L4 顶部 motif（10~20px 高）
     │       ║  翼/光晕/水晶    ║      │
y=8  │       ║                  ║      │
y=12 │       ╚══════════════════╝      │
     │        │    主体躯干     │        │  ← L3 主体（12~20px 高）
y=16 │        │                  │        │
y=20 │        │                  │        │
     │        ╔══════════════════╗      │
y=24 │        ║   细节/武器      ║      │  ← L2 特征（8~12px 高）
y=28 │        ╚══════════════════╝      │
     │       ╔════════════════════╗     │  ← L1 底座（8~12px 高）
y=32 │       ║   底座/地面        ║     │
y=36 │       ╚════════════════════╝     │
y=40 │                                   │
y=44 │                                   │  ← 安全边
y=48 │                                   │
y=52 │                                   │
y=56 │                                   │
y=60 │                                   │
     └──────────────────────────────┘
```

### 5 层结构（自下而上）

| 层 | y 范围 | 高度 | 内容 | 颜色优先级 |
|---|---|---|---|---|
| L1 base | 36~48 | 12px | 底座/地面阴影 | Dark + 灰 |
| L2 feature | 24~36 | 12px | 主体/躯干/武器 | Base 主色 |
| L3 detail | 16~24 | 8px | 细节/连接/铆钉 | Light + 灰 |
| L4 top | 4~16 | 12px | 顶部 motif（炮口/光环/水晶） | 主题 Accent |
| L5 frame | 0~4 | 4px | 顶部边缘/光晕 | Light + Accent |

| 底座主体 | x=12 y=52 w=40 h=4 | 居中 |

## Icon 骨架（32×32）

```
8x8 网格
y=0  ┌──────────────────┐
     │        ╔═══╗       │  ← 核心元素（4~6 格）
y=4  │        ║   ║       │
     │        ╚═══╝       │
y=8  │   装饰外圈/光环   │  ← 可选 1 格装饰
y=12 │                   │
y=16 │                   │  ← 安全边
     └──────────────────┘
```

### 3 层结构

| 层 | 内容 | 大小 |
|---|---|---|
| L1 core | 主体符号（箭头/星/雪花） | 16~20 px |
| L2 frame | 外圈/边框/对比 | 20~24 px |
| L3 halo | 装饰光点/小标记 | 2~4 px（点缀） |

## Tile 骨架（64×64）

瓦片需可平铺（seamless），边缘颜色需对称。

```
      左                  右
       ┌──────────────────┐
  y=0  │  A1  A2  A3  A4  │   ← 顶边色：与 y=60 镜像
       │                  │
       │   主纹理/格线     │   ← 中央主题
       │                  │
  y=60 │  D1  D2  D3  D4  │   ← 底边色：与 y=0 镜像
       └──────────────────┘
```

### 3 类

| 类型 | 主色 | 纹理元素 |
|---|---|---|
| buildable | 草绿（`#5a8c3a` 基底） | 网格线（4px 灰） |
| blocked | 暗灰（`#3a3a3a` 基底） | 裂纹（2px 黑） |
| path | 土棕（`#b8935a` 基底） | 路径纹理（4px 暗棕） |

## 颜色应用公式（以 corrosive 塔为例）

```xml
<!-- Dark = 腐蚀/金属底色 -->
<rect ... fill="#2a2a2a"/>

<!-- Base = 主体金属 -->
<rect ... fill="#4a4a4a"/>

<!-- Light = 边缘高光 -->
<rect ... fill="#7a7a7a"/>

<!-- Accent = 主题色（酸液） -->
<rect ... fill="#7fff00"/>

<!-- Accent Dark = 酸液阴影 -->
<rect ... fill="#3a7a00"/>
```

## 不要做的事

- ❌ 在 L4 顶部 motif 画大圆形（用 5x5 阶梯方块代替）
- ❌ 跨多层混合主题色（每层只 1~2 色）
- ❌ 在 32×32 图标内放超过 1 个核心元素
