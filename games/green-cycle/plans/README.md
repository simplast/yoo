## Findings considered and rejected

- **DPR 高分屏模糊**：Renderer.ts 用 `window.innerWidth/innerHeight` 未考虑 devicePixelRatio，但 `imageSmoothingEnabled=false` 保持像素感，且游戏是像素风设计，模糊感不显著。暂不处理，性价比低。
- **无尽模式排行榜混入普通通关记录**：Game.ts:479 `if (state.endless || won)` 让普通通关也记录到 endless 排行榜。这是设计选择（通关即"无尽第 31 波"），不算 bug。暂不改。
- **成长塔 6-30 级外推公式可能失衡**：HeroTower.ts:126-129 用 `1.1^(level-5)` 外推，30 级伤害约 563。但成长塔是终局主力，高伤害是设计意图。需实际游玩验证，暂不调整。
- **Audio 暂停时不停**：音效是程序化短音（事件触发即播完），无持续音频流，暂停时不会有残留。不需处理。
