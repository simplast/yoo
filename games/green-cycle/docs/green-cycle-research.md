# 《绿色循环圈》（Green Circle TD）调研报告

> 调研对象：基于《魔兽争霸 3：冰封王座》的国产经典自定义塔防地图《绿色循环圈》及其主要衍生版本（绿色循环圈 TD、绿色循环圈外传、外传 II/III、绿色循环圈海量版等）。
> 调研时间：2026-06-19
> 数据来源：KK 官方对战平台、游侠网/3DM、偶久网、天天 RPG、贴吧/斗鱼攻略、乐游参数表等公开资料（详见文末参考来源）。

---

## 0. 版本说明（必读）

“绿色循环圈”并非一张单一地图，而是一个持续演化了十余年的 **TD 地图系列**。不同版本在波数、资源、塔系、存档系统上差异较大，常见版本有：

| 版本                      | 代表作者           | 主要特点                                                                       |
| ------------------------- | ------------------ | ------------------------------------------------------------------------------ |
| 绿色循环圈 TD（原版/9.x） | 邪恶叔等           | 50 波经典循环圈，原始塔+成长塔+光环塔，护甲克制为核心                          |
| 绿色循环圈外传            | 六月 MAX / Froming | 引入成长塔、新护甲系数、PF 完美分、段位/成就/契约点                            |
| 绿色循环圈外传 II/III     | 佚名 / 涅槃        | PF 系统、龙 Boss、无尽排行榜、至尊点数                                         |
| 绿色循环圈海量版          | 萌王软             | 52 波传统/50 波次元、三国/西游/武侠/动漫等套装塔、装备合成、神龙许愿、时间宝石 |

本报告以 **“系列共性 + 版本差异”** 的方式整理，便于在网页像素风改编时取舍。

## 1. 游戏基本概述

### 1.1 玩法类型

- **类型**：多人合作循环圈塔防（Co-op Loop TD）。
- **引擎**：《魔兽争霸 3：冰封王座》自定义地图（.w3x）。
- **人数**：1–8 人（主流为 4 人或 8 人房）。
- **核心操作**：农民/建造者在草地上造塔 → 击杀沿循环路径移动的怪物 → 获取资源 → 升级/合成塔。

### 1.2 循环圈地图路径特点

- 地图通常由 **外圈 + 内圈** 两条首尾相连的正方形（或近似方形）圈道组成。
- 出怪口常见 **8 个**：外圈 4 个（位于四个边角），内圈 4 个（近似“卍”字形分布）。
- 所有圈道互相连通，怪物会 **无限循环** 直到被击杀，形成“上家漏怪 → 下家承接”的连锁防守关系。
- 玩家据点位于圈道外侧或内外圈之间的草地上，防御塔建在据点附近的可建造区域。

### 1.3 胜利与失败条件

#### 胜利条件（因版本而异）

- **原版 TD**：守住 50 波并击杀最终 Boss。
- **外传系列**：击杀第 50 波左右的 **龙 Boss**（骷髅/魔车/熊猫/龙等阶段性 Boss）。
- **海量版传统模式**：顶住 **52 波** 即可通关；次元模式则要求击败第 50 波最终 Boss。
- **无尽模式**：通关/达到指定 PF 分后开启，无法“通关”，以坚持波数/积分排名。

#### 失败条件

- **同屏怪物上限**：地图上同时存在的怪物数超过阈值即全员失败。常见阈值：
  - 普通模式：`游戏人数 × 40`
  - 无尽模式：`游戏人数 × 20`
- **Boss 超时**：特定 Boss 波（如奇美拉、黑龙、龙 Boss）未在规定时间内击杀，直接失败。
- **Perfect 分不足**：部分版本要求达到指定 PF 分才能进入无尽，未达成则视作未通关无尽。

### 1.4 波次敌人机制

- 每波怪物有预设的 **护甲类型、数量、移速、特殊能力**（隐形、空中、魔免、病毒免疫等）。
- 游戏界面右上角通常会提示下波属性，例如“多物理、中魔法、少神圣”。
- “多/中/少”大致对应数量：多≈75 只、中≈35 只、少≈15 只。
- 部分波次为 **Boss/精英波**，怪物血量极高或带光环，需要集中火力或特定塔克制。
- 每隔若干波会出现 **经济波/特殊波**（如兔子、飞艇），击杀后给予大量金币/木材。

## 2. 核心循环机制

### 2.1 循环路径

- 怪物从出生点进入圈道后，沿固定路线 **在外圈与内圈之间循环跑动**。
- 怪物 **不会自动消失**，只有被击杀才会离场。
- 若玩家防守点漏怪，怪物会进入下一位玩家的射程范围，形成“公共跑道”。
- 因此塔位选择不仅要覆盖自己的出怪口，还要考虑能否补刀其他玩家漏掉的怪。

### 2.2 怪物在圈内循环

- 循环路径是系列最核心的标识：与传统 TD 的“单行道到终点”不同，绿色循环圈的怪物会反复绕圈。
- 这种设计让 **减速、聚怪、控场** 的价值极高——把怪物留在高输出区越久，收益越大。
- 部分塔/技能可以主动改变怪物移动速度：
  - **驱逐塔**：增加范围内敌人移动速度（可让怪物更快进入主输出区，也更容易漏怪，需配合控制）。
  - **缓速塔 / 减速光环 / 小鬼 / 冰封**：降低怪物移速，实现聚怪。

### 2.3 生命值 / 漏怪惩罚

- 系列没有传统“基地生命值”概念，而是 **全局同屏怪物上限**。
- 漏怪本身不立即扣血，但会堆积在地图上；一旦总数超过上限并在倒计时内未降至阈值以下，全员失败。
- 单人模式下容错率最低；多人模式下上限按人数提升，但新手漏怪仍可能坑死全队。
- 部分版本还会因漏怪影响 PF（Perfect）分，进而影响无尽模式开启与排行榜积分。

### 2.4 收入 / 利息 / 人口机制

#### 收入

- **击杀奖励**：怪物被击杀时掉落金币，是主要收入来源。
- **波次奖励**：每波结束时根据击杀情况结算。
- **经济来源塔/道具**：
  - **神龙许愿塔**（海量版）：每 120 秒抽取一次物资奖励，可获金币、木材、人口等。
  - **工资卡**（海量版武器商店）：立即获得 10000 金币。
  - **成就商店兑换**：用成就点换金币/木材。

#### 利息

- 原版与传统外传系列 **没有典型的银行存款利息机制**。
- 海量版及某些魔改版本存在“自动加钱木”或利息脚本（如每 5 秒加 5%），但属于非官方魔改。
- 实际经济运营的核心是 **“滚雪球”**：前期少漏怪 → 多金币 → 早升神龙/成长塔 → 后期经济爆炸。

#### 人口 / 建造位

- 每座塔占用一定 **人口**（也叫建造位/人口上限）。
- 初始人口有限（例如外传 III 新手待遇为 8 人口），可通过消耗木材兑换人口扩展。
- 海量版中“人口”是重要资源，抽到人口等于大幅提升阵容上限。
- 部分塔有唯一性限制：
  - **成长塔/英雄塔**：通常每玩家只能建造 1 座（魔改版除外）。
  - **神器塔/悟吉塔**：全场唯一。

## 3. 塔类型与科技树

### 3.1 主要塔类别

根据《绿色循环圈 2.3 正式版》等经典划分，塔系通常分为 5 大类：

| 大类                 | 作用                               | 代表塔                                                                 |
| -------------------- | ---------------------------------- | ---------------------------------------------------------------------- |
| **成长塔（英雄塔）** | 主力输出，可升级、加属性点、学技能 | 天神、风神、电法、剑圣、将军、人马、寒冰、扫射、魔幻、巫妖、天将等     |
| **原始塔**           | 前期过渡或后期补伤害，造价相对固定 | 破坏塔、群攻塔、混乱塔、通灵塔、箭塔、闪电塔、毒塔、牛头塔、弹跳塔     |
| **辅助塔**           | 控场、增伤、削甲、单体爆发         | 驱逐塔、缓速塔、萨满塔、兽人塔、冰霜塔、腐蚀塔、重击塔、暴击塔         |
| **另类塔**           | 特殊机制、娱乐或过渡               | 魔石塔、科技塔、精灵塔、燃烧塔、炎魔塔、深渊塔、图腾塔、灵魂塔、死亡塔 |
| **光环塔**           | 范围 Buff/Debuff                   | 加速光环、加攻光环、减防光环、减速光环、冰封、火速、恐惧               |

### 3.2 升级路线与分支

#### 普通塔

- 通常有 **5–10 级** 基础等级，升到 Max 后可开启天赋或进化。
- 例如：毒塔 5 级 Max、图腾塔 5 级 Max、缓速塔满级后开启天赋可大幅提升减速。
- 海量版中，**光环塔** 可通过“天赋卷轴 / 天赋秘籍”开启基础/至尊天赋。

#### 成长塔（英雄塔）

- 每座成长塔有 **4 个技能**（部分塔更多），开局选择起手技能。
- 成长塔击杀怪物获得经验/蓝量，消耗资源升级，每次升级获得属性点。
- 属性点加在 **力量、敏捷、智力** 上，影响技能伤害、触发概率、攻速等。
- 常见属性节点：
  - 力量 80 / 160 / 200 / 240：解锁或增强技能
  - 智力 30 的倍数：提升某些技能伤害等级
  - 敏捷 300 左右：达到面板攻速上限
- 满级后可选择 **退功勋重造**（保留天赋效果）或继续堆属性。

### 3.3 特殊组合塔 / 合成配方

#### 海量版套装/组合示例

- **五虎将 → 魔神吕布**：将军塔 Max 后可走三国路线，最终合成魔神吕布。
- **君王组合**：拿破仑 + 亚历山大 + 铁木真 + 嬴政 → 解锁秦始皇（将军塔）。
- **东邪西毒南帝北丐**：武侠套装路线，常用于无尽减防（东邪减防）。
- **装备三合一**：通用装备通过宠物/背包进行三合一合成，部分版本新增隐藏合成。

#### 外传 III 帕鲁塔合成示例

- 帕鲁塔可召唤/合成多种单位，存在“白色兵种×4、黄色兵种×2”等数量要求。
- 单位分品质：白色（无等级需求）、绿色（帕鲁 5 级）、蓝色（帕鲁 20 级）、紫色（帕鲁 30 级）。
- 强化攻击/攻速对所有召唤单位共享；强化技能可出售返还等额金钱。

### 3.4 科技树与天赋

- **科技塔**：强化死亡塔、剑圣塔、图腾塔、神圣塔等特定塔。
- **神之科技研究所**（海量版）：部分英雄（如后裔）拥有独立科技，需消耗资源点满。
- **宠物天赋**：宠物可攻击并提供全局 Buff，例如小鬼增加巫妖伤害、鱼人增加渔人召唤概率等。
- **光环天赋**：Max 光环塔可花木材/成就购买天赋卷轴开启额外效果。

## 4. 资源系统

| 资源                | 主要用途                                       | 获取方式                                                           |
| ------------------- | ---------------------------------------------- | ------------------------------------------------------------------ |
| **金币**            | 造塔、升级塔、买塔基、买装备、买药水           | 击杀怪物、波次结算、成就兑换、工资卡、神龙许愿                     |
| **木材**            | 兑换人口、刷新随机塔、买天赋卷轴、兔子商店消费 | 击杀怪物、成就兑换、金币购买（部分版本 5000 金 = 10 木）、神龙许愿 |
| **人口 / 建造位**   | 限制同时存在的塔数量                           | 初始上限，木材兑换，神龙许愿抽到人口                               |
| **成就点**          | 购买局内道具、皮肤、特权                       | 局内成就、段位奖励、无尽积分换算                                   |
| **PF / Perfect 分** | 解锁无尽模式、计算排行榜                       | 每波开始前场上怪物为 0 则 +1                                       |
| **契约点 / 至尊点** | 契约主塔、解锁称号/皮肤                        | 提升段位/称号、达到指定无尽积分                                    |

### 4.1 金币

- 击杀怪物即时掉落，是所有塔升级的基础。
- 海量版中，开局第一波怪物金币极多，漏掉会显著影响发育。
- 塔出售可回收 **100% 建造资金**（升级资金不返还），因此前期可大量造 1 级塔补漏，后期卖掉换经济。

### 4.2 木材

- 木材是“高级货币”，主要用于人口扩展和随机/刷新。
- 部分版本可用金币直接购买木材，实现经济互转。
- 兔子商店 / 成就商店出售稀有道具、塔基、天赋卷轴，通常以木材计价。

### 4.3 人口 / 建造位

- 每座塔占用 1 点或多点人口。
- 成长塔、光环塔、辅助塔、套装塔共同竞争有限的人口。
- 高端局的阵容选择往往围绕“主塔 + 几个关键辅助 + 光环”优化人口配置。

### 4.4 科技升级

- 部分塔和科技建筑需要金币/木材进行研究。
- 海量版存在“神之科技”系列，需消耗资源将科技升到 20/25 级。
- 科技升级通常提升攻速、攻击范围、技能概率等全局或特定塔属性。

## 5. 特色系统

### 5.1 循环加速 / 速度控制

- **驱逐塔**：增加范围内敌人移动速度，常用于让怪物更快进入主输出区或配合聚怪。
- **缓速塔 / 减速光环 / 小鬼 / 冰封**：降低怪物速度，实现聚怪、延长输出时间。
- **加速光环 / 兽人 / 萨满**：提升友方塔攻速或攻击力，变相提高 DPS。
- **时间宝石**（海量版）：在无尽模式中使用，满足条件可直接提升 50 层无尽层数。

### 5.2 Boss 波

- 常见 Boss：奇美拉、黑龙、骷髅 Boss、魔车 Boss、熊猫 Boss、龙 Boss、绝望使徒等。
- Boss 波通常 **有时间限制**，超时未击杀直接失败。
- 对策：提前存钱、预留农民技能（神力一击/战争践踏）、切换攻击类型、使用全屏炸弹/全屏减速。

### 5.3 挑战怪 / 精英怪

- 部分版本加入 **精英怪、光环怪、专注怪** 等机制。
- 精英怪击杀后可获得属性点、金币、木材或装备。
- 光环怪会为周围怪物提供加速/回血/加攻等 Buff，需优先击杀或用神力一击秒杀。

### 5.4 合成配方

- **装备合成**：通用装备三合一，部分版本支持隐藏合成。
- **套装英雄合成**：如五虎将 → 魔神吕布、君王组合 → 秦始皇。
- **帕鲁塔单位合成**（外传 III）：通过不同品质单位组合出强力召唤物。

### 5.5 随机事件 / 随机机制

- **随机商店**：每局随机给玩家 11 个可建造塔（海量版），未随到想要的可用木材刷新。
- **神龙许愿塔**：每 120 秒随机抽取资源，是海量版最重要的经济来源之一。
- **兔子商店**：每局刷新的 NPC，出售药水、天赋卷轴、稀有塔基等。
- **挖宝 / 宠物**：部分版本可通过宠物或罗盘挖宝获得宝箱装备。
- **全屏炸弹 / 全屏减速**：紧急道具，用于救场。

### 5.6 难度 / 存档

- **难度分级**：简单/普通/困难（原版），部分版本还有 N1/N2 等难度。
- **平台存档**：
  - **段位**：青铜 → 白银 → 黄金 → 铂金 → 钻石 → 黑钻 → 大师 → 战神 → 王者 → 帝王。
  - **无尽积分**：记录最高无尽分数，每月可能清零。
  - **称号点 / 契约点 / 至尊点**：用于解锁称号、皮肤、主塔契约、成就商店商品。
- **新手保护**：外传 III 中地图等级 ≤11 级的新手开局获得 4000 金币、10 木材、8 人口。

## 6. 多人合作

### 6.1 玩家间互动

- **上下家关系**：每个玩家防守一个出怪口，漏怪会沿圈道流向下一家。
- **光环共享**：部分光环塔开启共享后，队友也能享受 Buff；站位时可将光环塔建在两玩家中间。
- **资源互助**：队友可赠送资源、共享塔基、互相补塔。
- **投票踢人**：遇到挂机或捣乱的玩家，可通过投票卷轴踢出。

### 6.2 漏怪共享 / 独立

- 怪物是 **全局共享** 的：漏怪不会消失，而是进入循环路径，所有玩家都有机会击杀。
- 失败条件也是 **全局共享** 的：同屏怪物上限全队共同承担。
- 因此新手漏怪会显著增加全队压力，老手常要求新手站在特定楼层（如外圈 1234、内圈 5678）并严格按攻略建塔。

### 6.3 组队策略要点

- **站位**：4 人局常用“左上 + 右下”对角站位，让漏怪有最长路径被补刀。
- **分工**：
  - 一人主 C（成长塔/套装塔）
  - 一人提供减速/缓速/精灵控怪
  - 一人提供加攻/加速/减防光环
  - 一人补伤害、补漏、应对特殊波
- **沟通**：提前告知下波护甲类型、Boss 波预留技能、共享光环开启等。

## 7. 网页像素风适配建议

若将《绿色循环圈》改编为网页像素风游戏，建议如下：

### 7.1 必须保留的核心机制

1. **循环路径**：外圈 + 内圈、怪物不死不消失、沿圈循环，这是“循环圈”的灵魂。
2. **护甲克制**：普通/穿刺/魔法/混乱 四种攻击类型对应 抗性/魔法/物理/神圣 四种护甲，必须保留并形成清晰 UI 提示。
3. **成长塔 + 技能树**：可升级、加属性点、学技能的成长塔是系列最大特色。
4. **全局漏怪上限**：用同屏怪物数上限替代传统基地血量，强化合作与紧张感。
5. **波次提示**：每波前显示护甲类型、数量、特殊能力。
6. **Boss 限时击杀**：保留关键 Boss 波的超时失败机制。
7. **多人合作（1–4 人或异步排行榜）**：保留上下家、光环共享、补刀等互动。

### 7.2 可简化的部分

1. **版本专属套装**：原版 50 波 + 经典塔即可，三国/西游/动漫套装可作为 DLC 或皮肤。
2. **复杂合成配方**：帕鲁塔、装备三合一可简化为一目了然的合成图谱，降低学习成本。
3. **平台存档段位**：可改为本地存档 + 赛季排行榜，减少平台依赖。
4. **农民微操**：神力一击、战争践踏等可简化为自动/半自动技能或固定 CD 大招按钮。
5. **随机商店**：网页版可保留“每局随机塔池”，但提供保底或刷新次数可视化。
6. **人口 / 木材互转**：可简化为“金币 = 唯一货币，人口 = 建造位上限”的二元经济。

### 7.3 像素表现要点

1. **地图**：用深绿色草地 + 深色圈道，突出“绿色”主题；外圈、内圈用不同灰度或格子区分。
2. **怪物**：
   - 用 16×16 或 24×24 像素精灵，按护甲类型上色（如神圣 = 金/白、物理 = 红、魔法 = 蓝）。
   - 隐形怪用半透明/闪烁效果；空中怪加阴影或飞行高度；Boss 用更大体型 + 血条。
3. **防御塔**：
   - 成长塔用不同职业造型（法师、战士、弓箭手），升级后改变颜色/光效。
   - 光环塔用地面光圈表现范围；减速/缓速用霜冻粒子；加速用黄色脉冲。
4. **攻击特效**：
   - 普通 = 白色弹丸；穿刺 = 箭矢；魔法 = 法球/闪电；混乱 = 紫黑色特效。
   - 暴击用红色跳字；眩晕用星星；中毒用绿色 dot。
5. **UI**：
   - 顶部显示金币、木材、人口、PF、当前波次、下波护甲。
   - 右侧/底部放置塔建造面板与技能按钮。
   - 漏怪/同屏怪物数用红色进度条直观提示。
6. **音效**：像素风可搭配 8-bit 攻击、升级、Boss 出现、漏怪警报音效。
7. **性能**：怪物数量多时应支持合并绘制、对象池、批量伤害计算，避免后期卡顿。
