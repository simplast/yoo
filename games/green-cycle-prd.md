# 《绿色循环圈》网页像素风塔防游戏 — 产品需求文档（PRD）

> 版本：v1.0
> 日期：2026-06-19
> 目标目录：`/Users/doer/dev/yoo/public/games/green-cycle/`
> 美术风格：轻微像素风（16×16 / 24×24 精灵），UI 不需要完全复刻原版《魔兽争霸 3》RPG 地图，但游戏机制与塔类型必须与系列经典设定保持一致。
> 平台：网页端（单文件 HTML/Canvas 或小型前端工程均可）。

---

## 1. 项目概述与目标

### 1.1 项目背景
《绿色循环圈》（Green Circle TD）是基于《魔兽争霸 3：冰封王座》的国产经典自定义塔防地图系列，核心特征是“怪物沿循环圈道无限跑动、不死不消失、上下家承接漏怪、全局同屏怪物上限”。本 PRD 旨在将该系列的核心玩法移植为网页端像素风塔防游戏，保留原作最具辨识度的循环路径、护甲克制、成长塔、合成与 Boss 挑战等机制，同时针对网页平台进行适度简化与现代化改造。

### 1.2 项目目标
1. **核心体验**：还原“循环圈”塔防的独特节奏——减速聚怪、补刀漏怪、Boss 限时rush、成长塔滚雪球。
2. **易上手难精通**：单局 40–60 分钟，50 波标准流程；新手可通关简单难度，高难度与无尽模式提供长期追求。
3. **轻量网页化**：优先单文件 HTML5/Canvas 或可构建为静态站点的小型工程，无需后端即可本地/在线运行。
4. **像素美术统一**：轻微像素风，强调可读性（护甲颜色、塔类型、技能范围），不追求完全复刻 War3 模型。

### 1.3 版本范围（MVP）
- **标准模式**：50 波 + 最终 Boss，支持 1–4 人本地/在线合作（MVP 可先做单人 + 异步排行榜）。
- **无尽模式**：通关标准模式后解锁，或主菜单直接选择“无尽练习”。
- **难度分级**：简单 / 普通 / 困难（影响怪物血量、金币奖励、同屏上限）。

### 1.4 目标用户
- 怀旧《绿色循环圈》原作的玩家。
- 喜欢循环路径、合作塔防、成长塔养成的中度策略玩家。
- 网页端休闲玩家（支持鼠标 + 键盘快捷键，可单手操作）。

### 1.5 成功指标
- 单局完整通关无阻塞、无崩溃。
- 50 波后期同屏 150+ 怪物仍保持 60 FPS（目标设备：2020 年后主流 PC / 中端手机）。
- 核心机制（循环、护甲克制、成长塔、漏怪上限）在首次游玩 10 分钟内可被玩家理解。

## 2. 核心玩法循环

### 2.1 地图路径设计
- **循环圈结构**：
  - 外圈：一个大正方形跑道，四角各有一个出怪口（共 4 个）。
  - 内圈：一个较小的正方形跑道，位于地图中央，近似“卍”字形或错位分布，4 个出怪口。
  - 内外圈通过 4 条连接道首尾相连，怪物可在外圈与内圈之间无限循环。
- **玩家据点**：玩家防御塔建在圈道外侧或内外圈之间的“草地”可建造区域上。4 人局常用对角站位（左上 + 右下），让漏怪有最长路径被补刀。
- **路径规则**：
  - 怪物进入跑道后沿固定方向循环，不会自动掉头或消失。
  - 塔不能建在跑道上，只能建在草地建造区。
  - 建造区用浅绿色/深绿色格子区分，鼠标悬停显示“可建造 / 不可建造”。

### 2.2 波次系统
- **标准模式**：共 50 波，每波间隔 10–20 秒（可手动加速）。
- **波次提示**：每波开始前 3 秒，顶部横幅提示下波信息：
  - 主要护甲类型（物理 / 魔法 / 神圣 / 抗性）。
  - 数量规模（多 / 中 / 少，对应约 75 / 35 / 15 只）。
  - 特殊能力（隐形 / 飞行 / 魔免 / 光环 / 病毒免疫等）。
- **经济波 / 特殊波**：
  - 每隔 5–7 波出现经济波（如兔子、飞艇），击杀后掉落大量金币或木材。
  - 经济波怪物血量较低、移速较快，适合全屏技能/爆发塔快速清场。

### 2.3 怪物行为
- 怪物出生后进入最近跑道节点，沿预设路径循环移动。
- **不击杀不消失**：漏掉的怪物会进入下一位玩家的射程，形成“上家漏怪 → 下家承接”的合作压力。
- 部分怪物带有能力：
  - **隐形**：仅真实/反隐塔或特定技能可攻击，否则穿塔而过。
  - **飞行**：不经过地面跑道，走独立空中路径，需对空塔。
  - **魔免**：免疫魔法伤害，仅物理/混乱攻击可造成伤害。
  - **光环怪**：为周围怪物提供加速 / 回血 / 加攻，需优先击杀。

### 2.4 漏怪与胜负条件

#### 2.4.1 胜利条件
- **标准模式**：守住 50 波并击杀第 50 波最终 Boss。
- **无尽模式**：无胜利条件，以坚持波数与 Perfect 分排名。
- **PF（Perfect）奖励**：每波开始时场上怪物数为 0，则 PF +1；PF 影响无尽解锁与排行榜积分。

#### 2.4.2 失败条件
- **同屏怪物上限**：地图上同时存在的怪物数超过阈值即全员失败。
  - 普通模式：上限 = 玩家数 × 40（单人 40，4 人 160）。
  - 无尽模式：上限 = 玩家数 × 20。
- **Boss 超时**：特定 Boss 波（如第 25、35、50 波）未在规定时间内击杀，直接失败。
- **紧急缓冲**：当同屏怪物达到上限的 80% 时，屏幕边缘红色闪烁并播放警报音效；达到 100% 时触发失败。

### 2.5 单局节奏
1. **前期（1–15 波）**：造原始塔过渡，尽量不漏怪，积累金币与木材，建造第一座成长塔。
2. **中期（16–35 波）**：成长塔升级、学技能、加属性点；补辅助塔（减速、减防、加攻光环）；应对第一次 Boss 波。
3. **后期（36–50 波）**：核心塔成型，处理魔免/飞行/光环怪海；预留资源与技能 CD；击杀最终 Boss。
4. **无尽（通关后）**：怪物属性指数增长，考验阵容上限与反应速度。

## 3. 玩家操作与输入

### 3.1 输入设备
- **鼠标（主要）**：
  - 左键：选择塔 / 建造塔 / 点击 UI 按钮。
  - 右键：取消建造 / 关闭面板。
  - 滚轮：缩放地图（可选，默认 1x）。
- **键盘（辅助）**：
  - `1–9` / `Q–R`：快速选择塔类型或技能（可自定义）。
  - `空格`：立即召唤下一波 / 结束当前波等待。
  - `Esc`：取消当前操作 / 打开暂停菜单。
  - `Shift + 左键`：连续建造同类型塔。
  - `Ctrl + 左键`：出售塔（需二次确认或长按）。
- **触屏（可选）**：
  - 单指拖拽移动视角。
  - 双指缩放。
  - 长按塔打开详情面板。

### 3.2 核心操作流程
1. **选择塔**：从底部/右侧塔面板点击要建造的塔。
2. **预览建造**：鼠标移到草地，显示塔的射程圈与像素投影。
3. **确认建造**：左键点击，扣除金币与人口，塔进入 0.5–1 秒建造动画。
4. **选择已建塔**：左键点击，弹出升级/出售/技能面板。
5. **升级**：点击升级按钮，扣除资源；满级后可选择进化/开启天赋/合成。
6. **出售**：返还 100% 建造资金，不返还升级资金（与原作一致）。

### 3.3 农民 / 建造者技能（简化）
- 原作中农民拥有“神力一击 / 战争践踏”等技能，网页版简化为全局救场按钮：
  - **神力一击**：CD 60 秒，对全屏造成一次巨额混乱伤害。
  - **全屏减速**：CD 90 秒，全屏怪物移速降低 50%，持续 8 秒。
  - **召唤支援**：CD 120 秒，在指定位置召唤 1 个临时高伤炮台，持续 15 秒。
- 按钮位于屏幕右下角，显示 CD 倒计时与像素图标。

### 3.4 镜头与视角
- 默认固定俯视 45° 或正俯视视角，整张地图可见。
- 支持小地图/缩略图点击快速跳转（无尽/大地图模式可选）。
- 战斗关键时刻（Boss 出现、漏怪警报）轻微镜头震动 + 红色边框。

## 4. 塔防系统

### 4.1 塔的分类总览
塔分为 5 大类，每类承担不同职责：

| 大类 | 定位 | 占用人口 | 代表塔 |
|------|------|----------|--------|
| 成长塔（英雄塔） | 主力输出，可升级、加属性点、学技能 | 2–3 | 天神、风神、电法、剑圣、将军、人马、寒冰、扫射、魔幻、巫妖、天将 |
| 原始塔 | 前期过渡或后期补伤害，造价固定 | 1 | 破坏塔、群攻塔、混乱塔、通灵塔、箭塔、闪电塔、毒塔、牛头塔、弹跳塔 |
| 辅助塔 | 控场、增伤、削甲、单体爆发 | 1–2 | 驱逐塔、缓速塔、萨满塔、兽人塔、冰霜塔、腐蚀塔、重击塔、暴击塔 |
| 另类塔 | 特殊机制、娱乐或过渡 | 1–2 | 魔石塔、科技塔、精灵塔、燃烧塔、炎魔塔、深渊塔、图腾塔、灵魂塔、死亡塔 |
| 光环塔 | 范围 Buff/Debuff | 1–2 | 加速光环、加攻光环、减防光环、减速光环、冰封、火速、恐惧 |

### 4.2 成长塔（英雄塔）
成长塔是游戏核心，每座塔拥有独立的属性、技能树与加点系统。

#### 4.2.1 基础规则
- **唯一性**：每玩家每局每种成长塔限造 1 座（魔改版无尽可放宽）。
- **经验获取**：击杀怪物获得经验；波次结算根据伤害/击杀额外奖励经验。
- **升级**：消耗金币升级，每次升级获得 1–3 个属性点。
- **属性点**：加在 **力量 / 敏捷 / 智力** 上：
  - 力量：提升技能伤害、生命值、部分触发概率。
  - 敏捷：提升攻击速度、暴击概率；约 300 敏捷达到面板攻速上限。
  - 智力：提升魔法伤害、技能等级；部分技能每 30 智力提升一级。
- **技能解锁**：
  - 每座成长塔开局选择 1 个起手技能。
  - 力量 80 / 160 / 200 / 240、智力 30 / 60 / 90 等节点解锁或增强技能。
- **满级选择**：
  - 退功勋重造：保留天赋效果，重置等级与属性点，重新加点（适合优化 build）。
  - 继续堆属性：直接提升面板，适合速通局。

#### 4.2.2 成长塔示例
| 成长塔 | 攻击类型 | 定位 | 代表技能 |
|--------|----------|------|----------|
| 天神 | 神圣 | 高面板单体/范围输出 | 神罚、审判之光、圣盾 |
| 风神 | 穿刺 | 高攻速、多目标 | 飓风、风刃、疾风步 |
| 电法 | 魔法 | 连锁闪电、范围控制 | 闪电链、雷暴、静电场 |
| 剑圣 | 普通 | 近战溅射、暴击 | 剑刃风暴、致命一击、分身 |
| 将军 | 普通/混乱 | 召唤/套装核心 | 冲锋、号令、军阵 |
| 人马 | 普通 | 肉盾型输出 | 践踏、反击、狂暴 |
| 寒冰 | 魔法 | 减速控制 | 冰霜新星、极寒领域 |
| 扫射 | 穿刺 | 持续单体爆发 | 扫射、穿透、连弩 |
| 魔幻 | 魔法 | 随机性高伤害 | 魔法飞弹、次元裂隙 |
| 巫妖 | 魔法 | 范围 DOT/减甲 | 霜冻新星、死亡凋零 |
| 天将 | 神圣 | 对空/对地全能 | 天兵降临、天雷 |

### 4.3 原始塔
- 造价低、建造快，是前期过渡与后期补伤害的主力。
- 通常有 5–10 级基础等级，升到 Max 后可开启天赋或作为合成材料。
- **关键原始塔**：
  - **破坏塔**：高面板单体普通攻击。
  - **群攻塔**：范围溅射，适合清理密集怪海。
  - **混乱塔**：混乱攻击，对所有护甲均有稳定伤害。
  - **通灵塔**：击杀后概率召唤骷髅，提供额外输出/肉盾。
  - **箭塔**：穿刺攻击，对低甲怪高效。
  - **闪电塔**：魔法链式攻击。
  - **毒塔**：持续毒伤（DOT），满级天赋可大幅提升。
  - **牛头塔**：高血量肉盾塔，可阻挡怪物前进（网页版可简化为减速/嘲讽）。
  - **弹跳塔**：弹射攻击，适合直线跑道。

### 4.4 辅助塔
- **驱逐塔**：增加范围内敌人移速，用于让怪物更快进入主输出区（高手向，配合减速）。
- **缓速塔 / 减速光环 / 冰封**：降低范围内怪物移速，实现聚怪。
- **萨满塔 / 兽人塔**：提升友方塔攻速或攻击力。
- **腐蚀塔 / 减防光环**：降低怪物护甲，提升团队输出。
- **重击塔 / 暴击塔**：提供概率眩晕 / 暴击加成。
- **冰霜塔**：范围减速 + 概率冻结。

### 4.5 光环塔
- 光环塔开启后，范围内友方塔或敌方怪物获得持续效果。
- **加速光环**：友方攻速 +15% / +25% / +35%。
- **加攻光环**：友方攻击力 +15% / +25% / +35%。
- **减防光环**：敌方护甲 -3 / -5 / -8。
- **减速光环**：敌方移速 -15% / -25% / -40%。
- **冰封**：大幅减速并概率冻结。
- **火速**：提升范围内怪物移速（与驱逐塔类似，偏特殊打法）。
- **恐惧**：概率使怪物短暂反向移动。
- **天赋系统**：光环塔 Max 后可消耗木材/成就点购买天赋卷轴，开启“基础天赋 / 至尊天赋”。

### 4.6 另类塔
- **魔石塔**：储存击杀能量，释放全屏/范围爆发。
- **科技塔**：强化特定塔（死亡塔、剑圣塔、图腾塔、神圣塔等）。
- **精灵塔**：提供回蓝/反隐/净化效果。
- **燃烧塔 / 炎魔塔**：持续火焰 DOT。
- **深渊塔 / 图腾塔 / 灵魂塔 / 死亡塔**：特效向，用于特定 build 或娱乐。

### 4.7 升级与进化
- **普通塔**：1 → 5 级（或 1 → 10 级），每级提升攻击力、攻速、射程。
- **成长塔**：1 → 30/50/100 级（不同塔上限不同），每次升级获得属性点。
- **进化**：部分塔 Max 后可消耗资源进化为高阶形态（例如毒塔 → 剧毒之源）。
- **天赋**：Max 后使用天赋卷轴开启额外被动。

### 4.8 合成 / 配方系统
网页版合成需通过清晰的“合成图谱”展示，降低学习成本。

#### 4.8.1 装备合成
- 通用装备通过三合一合成：3 件同等级装备 → 1 件高一级装备。
- 示例：3 件白装 → 1 件绿装 → 3 绿 → 1 蓝 → 3 蓝 → 1 紫 → 3 紫 → 1 橙。
- 部分装备支持隐藏合成：满足特定组合（如攻速 + 暴击 + 吸血）合成传奇装备。

#### 4.8.2 套装英雄合成
- **五虎将 → 魔神吕布**：将军塔 Max 后，场上同时存在关羽、张飞、赵云、马超、黄忠（或简化触发条件），可合成魔神吕布。
- **君王组合 → 秦始皇**：拿破仑 + 亚历山大 + 铁木真 + 嬴政（将军塔路线），解锁秦始皇。
- **武侠套装**：东邪 + 西毒 + 南帝 + 北丐，常用于无尽减防（东邪减防）。

#### 4.8.3 科技 / 宠物辅助合成
- **神之科技研究所**：部分英雄（如后羿）拥有独立科技树，消耗资源升到 20/25 级，提升攻速、范围、技能概率。
- **宠物**：宠物可攻击并提供全局 Buff，例如小鬼增加巫妖伤害、鱼人增加渔人召唤概率。

### 4.9 塔位与人口
- 每座塔占用 1–3 点人口。
- 初始人口：简单 12，普通 10，困难 8（单人数值）。
- 人口扩展：消耗木材兑换，每次 +1 人口，价格递增（如 5 木 → 10 木 → 20 木）。
- 网页版经济可简化为“金币 + 人口”二元模型，木材作为高级货币用于刷新/天赋/商店。

## 5. 敌人系统

### 5.1 普通怪物
- 按波次预设数量、血量、移速、护甲类型、特殊能力。
- 数量规模：多 ≈ 75 只，中 ≈ 35 只，少 ≈ 15 只。
- 随着波次推进，怪物血量与移速指数增长。
- 普通怪掉落金币，击杀即时到账。

### 5.2 精英怪 / 光环怪
- **精英怪**：体型略大，血量更高，击杀后掉落额外金币/木材/属性点/装备。
- **光环怪**：为周围怪物提供 Buff（加速、回血、加攻、加甲），优先级最高。
- **专注怪**：攻击时锁定最近塔，降低其攻速（可选机制）。

### 5.3 Boss 设计
Boss 波为关键压力点，通常有时间限制，超时未击杀直接失败。

| Boss | 出现波次 | 护甲类型 | 特点 | 对策 |
|------|----------|----------|------|------|
| 奇美拉 | 15 / 25 | 魔法/抗性 | 高血量、范围毒液 | 混乱塔 + 减防 + 神力一击 |
| 黑龙 | 25 / 35 | 神圣 | 飞行、魔免、高移速 | 对空神圣塔、全屏减速 |
| 骷髅 Boss | 20 | 物理 | 召唤小怪、回血光环 | 优先清理小怪、集火 Boss |
| 魔车 Boss | 30 | 抗性 | 高甲、免疫部分控制 | 腐蚀塔削甲、混乱输出 |
| 熊猫 Boss | 35 | 物理/魔法 | 高爆发、会跳怪 | 减速聚怪、预留爆发技能 |
| 龙 Boss | 45 / 50 | 神圣/混乱 | 最终 Boss，全屏技能 | 全阵容输出 + 全屏救场 |
| 绝望使徒 | 无尽模式 | 混乱 | 属性随层数增长 | 极限 build 与操作 |

- **Boss 计时器**：Boss 出现后屏幕上方出现倒计时条（如 60 秒 / 90 秒）。
- **Boss 血条**：Boss 头顶显示独立血条，受攻击时显示伤害跳字。

### 5.4 护甲类型
网页版保留 4 种护甲类型，并用颜色/图标直观区分：

| 护甲类型 | 颜色 | 图标 | 特点 |
|----------|------|------|------|
| 物理护甲 | 红色 | 盾牌 | 受穿刺/普通攻击影响大，魔法伤害较低 |
| 魔法护甲 | 蓝色 | 法袍 | 受魔法攻击影响大，物理伤害较低 |
| 神圣护甲 | 金色/白色 | 圣徽 | 仅混乱/神圣攻击高效，其他类型伤害极低 |
| 抗性护甲 | 紫色 | 龙鳞 | 对普通/穿刺/魔法均有一定减免，混乱稳定 |

### 5.5 攻击类型与克制

| 攻击类型 | 代表塔 | 对物理甲 | 对魔法甲 | 对神圣甲 | 对抗性甲 |
|----------|--------|----------|----------|----------|----------|
| 普通 | 破坏塔、剑圣 | 100% | 75% | 25% | 75% |
| 穿刺 | 箭塔、风神 | 150% | 75% | 25% | 75% |
| 魔法 | 电法、巫妖 | 75% | 150% | 25% | 75% |
| 混乱 | 混乱塔、魔神吕布 | 100% | 100% | 100% | 100% |
| 神圣 | 天神、天将 | 100% | 100% | 200% | 100% |

- 克制关系需在波次提示、塔详情面板、怪物血条旁直观展示。
- 建议 UI 使用箭头图标：绿色向上箭头表示克制，红色向下箭头表示被克制。

### 5.6 特殊能力
- **隐形**：半透明/闪烁效果，仅反隐塔、精灵塔、部分技能可攻击。
- **飞行**：走空中路径，地面塔无法攻击，需对空塔或神圣/魔法塔。
- **魔免**：免疫魔法伤害，仅物理/混乱攻击有效。
- **病毒免疫**：免疫毒/持续伤害，需直接伤害击杀。
- **加速**：移速提升，需要更多减速控制。
- **分裂**：死亡时分裂为多个小怪。

## 6. 资源与经济系统

### 6.1 资源类型

| 资源 | 用途 | 获取方式 | 备注 |
|------|------|----------|------|
| 金币 | 造塔、升级塔、买装备、买药水 | 击杀怪物、波次结算、成就兑换、工资卡、神龙许愿 | 主要货币 |
| 木材 | 兑换人口、刷新随机塔、买天赋卷轴、兔子商店消费 | 击杀怪物、成就兑换、金币购买、神龙许愿 | 高级货币 |
| 人口 | 限制同时存在的塔数量 | 初始上限、木材兑换、神龙许愿 | 核心建造限制 |
| 成就点 | 购买局内道具、皮肤、特权 | 局内成就、段位奖励、无尽积分换算 | 局外成长 |
| PF 分 | 解锁无尽模式、计算排行榜 | 每波开始前场上怪物为 0 则 +1 | 技巧分 |
| 契约点/至尊点 | 契约主塔、解锁称号/皮肤 | 提升段位/称号、达到指定无尽积分 | 长期追求 |

### 6.2 金币经济
- 击杀怪物即时掉落金币，是主要收入来源。
- 波次结束时根据击杀/漏怪情况结算额外金币。
- 网页版建议保留“开局第一波金币极多”的设定，鼓励玩家前期不漏怪。
- 塔出售返还 100% 建造资金，升级资金不返还，鼓励前期灵活补塔。

### 6.3 木材经济
- 木材用于人口扩展、刷新随机商店、购买天赋卷轴与稀有道具。
- 部分版本可用金币购买木材（如 5000 金 = 10 木），网页版可保留该兑换入口，但限制次数以防经济失衡。

### 6.4 人口管理
- 每座塔占用 1–3 人口。
- 成长塔、光环塔、辅助塔共同竞争有限人口，是主要策略维度。
- 高端局阵容通常围绕“1–2 座主塔 + 2–3 个关键辅助 + 1–2 个光环”优化。

### 6.5 利息机制（简化）
- 网页版可不保留复杂的银行存款利息。
- 经济运营核心仍是“滚雪球”：前期少漏怪 → 多金币 → 早升神龙/成长塔 → 后期经济爆炸。
- 可考虑加入“连胜奖励”：连续多波 PF +1 获得额外金币/木材。

### 6.6 商店系统
- **随机塔商店**：每局随机给玩家 11 个可建造塔，未随到想要的可用木材刷新。
- **兔子商店**：每局刷新 NPC，出售药水、天赋卷轴、稀有塔基、装备宝箱。
- **成就商店**：用成就点兑换金币/木材/皮肤/特权。

## 7. 特色系统

### 7.1 神龙许愿塔
- **定位**：海量版最重要的经济来源之一，网页版作为核心特色塔保留。
- **建造**：消耗一定金币/木材建造，占用 1 人口。
- **机制**：每 120 秒自动抽取一次物资奖励，随机获得以下一项：
  - 大量金币（如 2000–10000）
  - 木材（5–20）
  - 人口 +1
  - 稀有装备宝箱
  - 全屏炸弹 / 全屏减速救场道具
  - 属性点（成长塔直接获得经验）
- **许愿动画**：像素龙盘旋动画 + 奖励跳字。
- **策略**：尽早建造神龙许愿塔可显著加速经济，但占用 1 人口与前期资源，需权衡。

### 7.2 Boss 挑战
- 关键 Boss 波（15、25、35、45、50）触发全屏提示与倒计时。
- Boss 击杀后掉落大量资源与装备，失败则直接游戏结束。
- 玩家可提前手动“召唤 Boss”（跳过等待）以获取时间奖励或挑战更高 PF。

### 7.3 加速与速度控制
- **游戏速度**：支持 1x / 2x / 3x 加速（Boss 波与危险状态下限制最高 2x）。
- **驱逐塔**：增加范围内敌人移速，用于让怪物更快进入主输出区。
- **缓速塔 / 减速光环 / 小鬼 / 冰封**：降低怪物速度，实现聚怪。
- **加速光环 / 兽人 / 萨满**：提升友方塔攻速或攻击力，变相提高 DPS。
- **时间宝石**（无尽模式）：满足条件可直接提升 50 层无尽层数（可选后期内容）。

### 7.4 随机商店与兔子商店
- **随机塔池**：每局开始时随机决定本局可建造的塔列表，玩家可用木材刷新。
- **兔子商店**：固定时间出现，出售限时稀有道具：
  - 药水：回复塔生命、临时提升攻速/攻击。
  - 天赋卷轴：开启光环/成长塔天赋。
  - 稀有塔基：解锁本局未随到的塔。
  - 装备宝箱：随机品质装备。

### 7.5 全屏救场道具
- **全屏炸弹**：造成一次巨额混乱伤害，CD 长。
- **全屏减速**：全屏怪物移速大幅降低，持续数秒。
- **金币雨**：立即获得大量金币（消耗型道具）。
- 道具可通过神龙许愿、兔子商店、成就奖励获得。

### 7.6 成就与段位
- **局内成就**：
  - 首次不漏怪通关一波：+成就点。
  - 单波击杀 100 只怪物。
  - 首次合成魔神吕布 / 秦始皇。
  - 无尽模式达到 100/200/500 层。
- **段位系统**（局外）：
  - 青铜 → 白银 → 黄金 → 铂金 → 钻石 → 黑钻 → 大师 → 战神 → 王者 → 帝王。
  - 提升段位解锁称号、皮肤、主塔契约、成就商店商品。
- **无尽积分**：记录最高无尽分数，每月/每赛季可能清零或衰减。

### 7.7 难度与新手保护
- **难度分级**：
  - 简单：怪物血量 -20%，金币 +20%，同屏上限 +10，适合新手。
  - 普通：标准参数。
  - 困难：怪物血量 +30%，金币 -10%，同屏上限 -10，适合高手。
- **新手保护**：地图等级较低的新手开局获得额外金币、木材、人口（可选）。

### 7.8 多人合作（可选扩展）
- 上下家关系：每个玩家防守一个出怪口，漏怪流向下一家。
- 光环共享：部分光环塔可开启共享，队友也能享受 Buff。
- 资源互助：队友可赠送资源、共享塔基、互相补塔。
- MVP 可先实现本地多人热座或异步排行榜，后续扩展为在线合作。

## 8. UI/UX 与像素风设计规范

### 8.1 美术风格
- **整体风格**：轻微像素风，16×16 或 24×24 基础精灵，2D 俯视视角。
- **色调**：以深绿色草地为主，搭配深色跑道与亮色 UI，呼应“绿色循环圈”主题。
- **可读性优先**：怪物护甲颜色、塔类型、技能范围必须一目了然。

### 8.2 地图视觉
- **草地**：深绿/浅绿交替的像素格子，可建造区用浅绿色，不可建造区用深绿色或棕色。
- **跑道**：灰色/深棕色砖块路径，外圈与内圈用不同纹理或颜色区分。
- **出怪口**：黑色/红色漩涡门，Boss 波发出红光。
- **玩家据点**：每个玩家区域用不同颜色边框标识（4 人局：红/蓝/绿/黄）。

### 8.3 怪物视觉
- 基础尺寸 16×16，精英 24×24，Boss 48×48 或更大。
- 按护甲类型上色：
  - 物理护甲：红色调
  - 魔法护甲：蓝色调
  - 神圣护甲：金/白色调
  - 抗性护甲：紫色调
- 特殊能力表现：
  - 隐形：半透明 + 闪烁。
  - 飞行：带阴影、略高于地面。
  - Boss：独立血条、发光效果、入场动画。
  - 光环怪：脚下持续光环。

### 8.4 防御塔视觉
- 成长塔：用不同职业造型（法师、战士、弓箭手），升级后改变颜色/光效。
- 光环塔：地面显示范围光圈，颜色对应光环类型（黄色加速、红色加攻、蓝色减速、紫色减防）。
- 辅助塔：范围效果用粒子表现（霜冻、闪电、火焰、毒气）。
- 建造预览：半透明投影 + 射程圈。

### 8.5 攻击特效
- 普通：白色弹丸。
- 穿刺：箭矢或飞镖。
- 魔法：法球、闪电、冰锥。
- 混乱：紫黑色特效。
- 神圣：金色光束。
- 暴击：红色跳字 + 闪光。
- 眩晕：头顶星星。
- 中毒：绿色 dot + 血条变绿。

### 8.6 UI 布局
- **顶部状态栏（从左到右）**：
  - 金币、木材、人口（当前/上限）
  - 当前波次 / 总波次
  - 下波护甲类型图标与文字
  - PF 分
  - 同屏怪物压力条（绿/黄/红）
- **底部/右侧塔面板**：
  - 成长塔、原始塔、辅助塔、光环塔、另类塔分页。
  - 选中塔显示造价、人口、攻击类型、射程。
- **右下角**：全局技能按钮（神力一击、全屏减速、召唤支援）。
- **左下角**：小地图/缩略图（可选）。
- **波次提示**：屏幕中央上方横幅，显示下波护甲、数量、特殊能力。
- **Boss 提示**：屏幕中央大字号“BOSS 来袭” + 倒计时条。

### 8.7 交互反馈
- 建造成功：塔拔地而起 + 音效。
- 升级成功：金光一闪 + 音效。
- 漏怪警报：屏幕边缘红色闪烁 + 警报音效。
- Boss 出现：镜头轻微震动 + 特殊 BGM。
- 击杀跳字：金币 +XX、暴击 -XXX、神圣 -XXXX。

### 8.8 音效与音乐
- **BGM**：像素风循环背景音乐，Boss 波切换紧张战斗音乐。
- **音效**：
  - 建造：8-bit 放置声。
  - 升级：8-bit 升级声。
  - 攻击：不同攻击类型对应不同音效（箭矢、闪电、爆炸、圣光）。
  - Boss 出现：低沉警报。
  - 漏怪：高频警告声。
  - 通关：胜利音乐。
  - 失败：低沉结束音。

### 8.9 字体与文案
- 使用像素字体（如 Zpix、VT323、Press Start 2P 中文适配版）。
- 数字、血量、伤害跳字使用等宽像素字体，确保对齐。

## 9. 性能与兼容性要求

### 9.1 技术选型建议
- **渲染**：HTML5 Canvas 2D（优先）或 WebGL（如怪物数量极大时）。
- **框架**：原生 JS / TypeScript + Canvas，或 Phaser 3 / PixiJS。
- **构建工具**：Vite / Rollup，输出为静态站点。
- **存储**：LocalStorage 保存本地存档、设置、排行榜。
- **多人（后续）**：WebSocket 或 WebRTC。

### 9.2 性能目标
- **帧率**：目标 60 FPS，最低 30 FPS。
- **怪物上限**：标准模式 160 只，无尽模式支持 300+ 只同屏。
- **优化策略**：
  - 对象池：怪物、子弹、特效复用，避免频繁 GC。
  - 合并绘制：同类型怪物批量渲染或 Sprite Batching。
  - 空间分区：四叉树 / 网格加速碰撞与索敌。
  - 伤害批处理：同一帧内对同一目标的多次伤害合并显示。
  - LOD：远距离怪物简化动画，非可见区域暂停更新。
  - 限制特效数量：同屏粒子数上限，超出时优先保留高优先级特效。

### 9.3 兼容性
- **浏览器**：Chrome / Edge / Firefox / Safari 最新两个主版本。
- **分辨率**：
  - 桌面端：1280×720 及以上，支持窗口化与全屏。
  - 移动端：720p 及以上，支持触屏与虚拟按钮（可选）。
- **输入**：鼠标 + 键盘必备；触屏适配为可选增强。
- **离线**：单文件 HTML 可在无网络环境下本地打开（除多人功能外）。

### 9.4 数据持久化
- 本地存档：难度解锁、成就、段位、无尽最高分、设置。
- 导出/导入存档：支持 JSON 文件导出与导入，方便玩家迁移。
- 防作弊：排行榜数据如需联网，需后端校验；本地模式不做严格反作弊。

## 10. 后续可扩展点

### 10.1 多人合作
- **本地热座**：多玩家共用一台设备，轮流操作或分屏。
- **在线合作**：2–4 人实时联机，保留上下家、光环共享、资源互助、投票踢人。
- **异步排行榜**：无尽模式分数上传服务器，按周/赛季排名。

### 10.2 存档与成长
- **云存档**：跨设备同步进度。
- **赛季系统**：每月刷新无尽积分，发放赛季奖励。
- **成就系统扩展**：更多隐藏成就、称号、头像框。

### 10.3 更多塔与内容
- **版本专属套装**：三国（五虎将 / 吕布）、西游、武侠、动漫主题塔作为 DLC 或赛季内容。
- **新成长塔**：每月新增 1–2 座成长塔，丰富 build。
