一份博客写作时常用的 Markdown 语法速查表。

## 标题

```markdown
## 二级标题
### 三级标题
#### 四级标题
```

## 文本样式

```markdown
**加粗文本**
*斜体文本*
~~删除线~~
`行内代码`
```

效果：**加粗文本**、*斜体文本*、~~删除线~~、`行内代码`

## 列表

无序列表：

```markdown
- 项目一
- 项目二
  - 嵌套项目
```

有序列表：

```markdown
1. 第一项
2. 第二项
```

## 链接与图片

```markdown
[链接文字](https://example.com)
![图片说明](./image.png)
```

## 代码块

\`\`\`javascript
function hello() {
  console.log('Hello, World!');
}
\`\`\`

## 引用

```markdown
> 这是一段引用文字。
> 可以有多行。
```

## 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
```

## 分割线

```markdown
---
```

更多语法请参考 [Markdown Guide](https://www.markdownguide.org/)。
